🔧 MANUAL SETUP (If you don't want to use setup script)

1. Copy config:
   cp config.example.json config.json

2. Create API key:
   echo "PROXY_API_KEY=$(openssl rand -hex 16)" > .env

3. Start service:
   docker compose up -d

4. Wait 1-2 minutes, then test:
   API_KEY=$(grep PROXY_API_KEY .env | cut -d= -f2)
   curl -H "X-Api-Key: $API_KEY" http://localhost:8083/stat

📋 USEFUL COMMANDS

# View logs
docker compose logs -f proxy-checker

# Restart service
docker compose restart proxy-checker

# Check health
curl http://localhost:8083/health

# Get your API key
cat .env | grep PROXY_API_KEY

# Get statistics
API_KEY=$(grep PROXY_API_KEY .env | cut -d= -f2)
curl -H "X-Api-Key: $API_KEY" http://localhost:8083/stat | jq

# Get a proxy
curl -H "X-Api-Key: $API_KEY" http://localhost:8083/get-proxy

See QUICKREF.md for more commands.

🌐 API ENDPOINTS

Port: 8083 (changed from 8080 in this version)

  • http://localhost:8083/health        - Health check
  • http://localhost:8083/stat          - Statistics (auth required)
  • http://localhost:8083/get-proxy     - Get proxy (auth required)
  • http://localhost:8083/metrics       - Prometheus metrics
